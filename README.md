# Tourism Booking Analysis

## Overview

This project analyzes hotel booking demand data to understand booking behavior, cancellations, lead time, seasonality, and customer segments.

---

## Business Problem

The goal of this project is to turn raw data into practical business insights. The analysis focuses on identifying patterns, measuring performance, and recommending actions that can improve decision-making.

---

## Dataset

**Dataset used:** Hotel Booking Demand Dataset + optional World Bank tourism arrivals

## Key Questions

- What percentage of bookings are canceled?
- Which hotel type has the highest cancellation rate?
- How does lead time affect cancellation?
- Which months have the highest demand?
- Which customer segments generate the strongest booking volume?
